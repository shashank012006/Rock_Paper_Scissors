# Rock Paper Scissors

A simple Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Features

* Play Rock, Paper, Scissors against the computer
* Random computer choices every round
* 5-round match system
* Live score tracking
* Displays round winner and final winner
* Reset game functionality

## How to Play

1. Click one of the three buttons:

   * ✊ Rock
   * ✋ Paper
   * ✌️ Scissors
2. The computer randomly selects its move.
3. The winner of each round is displayed.
4. After 5 rounds, the final winner is announced.
5. Click **Reset Game** to start a new match.

## Winning Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Same choices result in a tie
